1. Download the zip and extract all files
2. Import maven project into IDE(Eclipse, IntelliJ, etc.)
3. Run the 'testng.xml' located at path - contiq\src\test\Resources\contiq\testrunners 
4. To view Extent report for current execution, open 'TestExecutionReport.html' file present under - contiq\build\ 
5. In case test fails, you can refer a screenshot under path- '\contiq\screenshots'
